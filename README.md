# Customer Churn Prediction Project
Customer Churn Analysis (Data Cleaning &amp; EDA) A data analysis project focusing on cleaning and exploring customer churn dataset to understand patterns and key factors leading to customer churn

##  Project Overview
This project focuses on predicting customer churn.
The dataset is messy and requires proper cleaning and preprocessing.

Source: Kaggle  
Link: https://www.kaggle.com/datasets/ankitverma2010/ecommerce-customer-churn-analysis-and-prediction
